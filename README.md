# 🌌 StarSight AI

StarSight AI is an AI-powered constellation identification system that detects stars from night-sky images and identifies constellations using Computer Vision and Machine Learning.

## Features

- Star Detection using OpenCV
- Constellation Recognition
- KNN Machine Learning Model
- Template Matching
- Dark Theme GUI (PySide6)
- Voice Narration Support
- Constellation Visualization

## Technologies Used

- Python
- PySide6
- OpenCV
- NumPy
- Scikit-learn
- pyttsx3

## Project Structure

```text
app.py
star_engine.py
constellation_data.py
train.py
constellation_model.npz
```

## Run

```bash
pip install -r requirements.txt
python app.py
```
